# Mocha + Chai + Selenium + NodeJS


## Pre-requisites
* Node
* VScode 


### Installation

```
npm install 

npm install -g mocha
```


### Running test locally
```
npm run test
```

### Running test in CI/CD with Reports and API
```
When Runnin
To run all test please run with Reports
./run.sh

To run specific test please run
./run.sh Test-2

```
